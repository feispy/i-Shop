# i-Shop
  Online store on Python
### 1. Installation

```bash
$ git clone https://github.com/feispy/i-Shop.git
$ cd i-Shop
$ pip install -r requirements.txt
```

### 2. Running The App

```bash
python main.py
```

### 3. Viewing The App

Go to `http://127.0.0.1:5000`

### 4. Technology Stacks
- flask
- flask-sqlalchemy
- bootstrap
